# ☕ Система управления кафе

Профессиональная веб-система для управления кафе с ролевой моделью доступа.

## 🚀 Функционал

- ✅ **Авторизация и регистрация** пользователей
- ✅ **Ролевая модель**: Кассир, Менеджер, Администратор
- ✅ **Создание заказов** с выбором блюд
- ✅ **Управление меню** (добавление, редактирование, удаление)
- ✅ **Статусы заказов** (новый, готовится, готов, завершен)
- ✅ **Строгий дизайн** в корпоративном стиле

## 📋 Требования

- Python 3.8+
- Flask 3.0+

## 🛠️ Установка

```bash
# Клонирование репозитория
git clone https://github.com/ВАШ_НИК/cafe-management-system.git
cd cafe-management-system

# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (Mac/Linux)
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python app.py
